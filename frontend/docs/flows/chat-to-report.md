# Flow: Chat to Report

1. User opens dashboard.
2. User writes free-form idea.
3. No preview appears while typing.
4. User clicks `Susun brief`.
5. App extracts brief fields.
6. If incomplete: show preview + missing state + CTA `Lengkapi informasi`.
7. User enters clarification answers.
8. CTA enabled when required fields filled.
9. App shows processing screen with skeleton.
10. App opens report.
11. User can open Full Report or Slide Deck.
