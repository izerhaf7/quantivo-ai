# Report Summary

Purpose: concise business analysis output.

Content modules:
- Executive summary / top insight
- Score/confidence
- SWOT
- Competitor/market notes
- Prioritized actions
- Sentiment distribution
- Key metrics
- Evidence/assumption confidence

Design:
- Readable dossier style.
- Dense but not cramped.
- Dark/light contrast strong.
- Primary actions: Full Report, Slide Deck, New Analysis.
